# 무안 평산교회 웹사이트 제작 계획서

## 📋 프로젝트 개요

**목표**: 따뜻하고 신뢰감 있는 교회 웹사이트 제작
- **타겟**: 새신자/이주민, 기존 성도, 외부인
- **핵심 가치**: 화려함보다 따뜻함과 정보 전달
- **배포**: Netlify (정적 웹사이트)

---

## 🎯 1단계: 기술 스택 결정

### 선택된 기술 스택
- **Framework**: Next.js (App Router) - Netlify와 완벽 호환, SEO 최적화
- **Styling**: Tailwind CSS - 빠른 개발, 반응형 디자인 용이
- **Animation**: Framer Motion - 부드러운 스크롤 애니메이션
- **배포**: Netlify - 무료 호스팅, 자동 배포

### 선택 이유
- pilotplan의 실용성 + reference_prompt의 세련된 디자인 결합
- 모바일 최우선 설계 가능
- 유지보수 용이 (컴포넌트 기반)

---

## 📐 2단계: 사이트 구조 (Single Page Application)

### 섹션 구성 (스크롤 방식)

1. **Hero Section (최상단)**
   - 풀스크린 배경 이미지 (교회 전경 또는 따뜻한 예배당 내부)
   - 환영 메시지 (대형 타이포그래피)
   - 부드러운 스크롤 인디케이터

2. **교회 소개 (About)**
   - 담임목사님 인사말 및 프로필 사진
   - 교단 마크 (기독교대한성결교회)
   - 목회 철학 / 교회 연혁

3. **예배 안내 (Worship Times)** ⭐ 가장 중요
   - 주일예배, 수요예배, 새벽예배 시간표
   - Bento Grid 스타일 카드 레이아웃

4. **오시는 길 (Location)**
   - 네이버/카카오 지도 연동
   - 주차 정보, 셔틀 운행 여부
   - 상세한 길 안내 (시골길 특성 고려)

5. **설교/갤러리 (Sermon & Gallery)**
   - 유튜브 링크 연동 (최신 설교)
   - 행사 사진 갤러리

6. **연락처/Footer**
   - 전화번호, 주소
   - 헌금 계좌 정보 (필요시)

## 🎨 3단계: 디자인 시스템

### 컬러 팔레트
- **베이스**: 오프 화이트 (#f5f5f7) - 따뜻한 느낌
- **텍스트**: 짙은 회색 (#1d1d1f) - 부드러운 검정
- **강조색**: 
  - 보라색 계열 (기성 교단 상징색) - 포인트 컬러
  - 채도 낮은 골드/세이지 그린 - 버튼/강조 요소
- **배경**: 화이트 & 우드(베이지) 톤

### 타이포그래피
- **폰트**: Pretendard (한글 최적화) 또는 Noto Sans KR
- **크기**: 
  - 헤드라인: 48px~72px (모바일: 32px~48px)
  - 서브 헤드라인: 24px~32px
  - 본문: 16px~18px (어르신 가독성 고려)
- **계층**: 명확한 타이포그래피 계층 구조

### 레이아웃 원칙
- **여백**: 넓은 여백 (Gutter, Padding) - 시원하고 여유로운 느낌
- **카드**: 둥근 모서리 (border-radius: 12px~16px)
- **그리드**: Bento Grid 스타일 (Apple 스타일 차용)

### 인터랙션
- **스크롤**: Fade Up 애니메이션
- **호버**: 부드러운 스케일/그림자 효과
- **네비게이션**: 글래스모피즘 (Backdrop blur) - 스크롤 시 고정

```

├── app/                    # Next.js App Router
│   ├── layout.tsx         # 루트 레이아웃

│   └── globals.css        # 전역 스타일
├── components/             # 컴포넌트 모듈
│   ├── Hero.tsx           # 히어로 섹션
│   ├── About.tsx          # 교회 소개
│   ├── WorshipTimes.tsx   # 예배 안내
│   ├── Location.tsx       # 오시는 길
│   ├── SermonGallery.tsx  # 설교/갤러리
│   ├── Footer.tsx         # 푸터
│   └── Navigation.tsx     # 네비게이션 바
├── public/                # 정적 파일
│   ├── images/            # 이미지 파일
│   └── ...
├── references/            # 참고 문서
├── pictures/              # 원본 사진
├── .cursorrules           # Cursor AI 지침
└── package.json
```
